Font Design

A custom display typeface that shapes the Latin (English) alphabet to evoke Meetei Mayek — so the title, logo, and UI feel unmistakably Manipuri while staying readable to a global player.

TL;DR

Design an English display face whose letterforms borrow the visual DNA of Meetei Mayek — its rounded bowls, looping strokes, and distinctive terminals — without becoming illegible or a caricature. It is for titles, headings, and signage, not body text. The real script is used for genuine Meiteilon; this face is the flavour layer on top.

Evoke, don't counterfeit

This is a Latin face inspired by Meetei Mayek — not a replacement for the script and not "fake writing." Actual Meiteilon text must be set in genuine Meetei Mayek (see Historical Research on language/script). The design must respect the script, never mock it; review with Cultural Integrity.

Design goals

Evocative
A first glance reads "Manipur / Meetei Mayek," even to someone who can't read the script.
Legible
English stays instantly readable — every A is an A. Character, not confusion.
Display-first
Tuned for large sizes: title screen, logo, section headers, in-world signage.
Respectful
Borrows real structural features of the script, not a parody of them.

The idea in one picture

Take the skeleton of each Latin letter and reshape it with Meetei Mayek's rounded, looping character — keeping the letter recognisable.

Where it's used — and where it isn't

Use it for

Game title & logo, chapter/section headers, in-world signage, splash and menu accents.

Pair it with

A clean, neutral sans for body text and UI, so long strings stay readable and accessible.

Don't use it for

Paragraphs, subtitles, or anything a player must read quickly — and never to fake real Meiteilon.

Feature checklist

FeatureTarget

WeightsAt least Regular + Bold for title/heading hierarchy
FormatWeb font (WOFF2) + source (glyphs/UFO) kept out of the repo if heavy
AccessibilityDisplay-only; body text uses a paired legible sans
