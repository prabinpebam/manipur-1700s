How to burn it down

  1. Pick a detail document

    Open the owning domain overview and choose a "Planned" card, respecting priority order.

  2. Author it as a Slate page

    Add a new .html/.md file in the domain folder, lead with a TL;DR, and visualise any data. Register it in docs-manifest.json.

  3. Update status

    Flip the card and this tracker from Drafting toward Complete as review passes.

Definition of done

A detail document is "Complete" only when every load-bearing claim traces to a High-confidence entry in the Historical Database. See How We Work.
